# OverTheWire Natas Walkthrough


This repository documents my progress through the Natas wargame from OverTheWire.

Natas is focused on introductory to intermediate web security concepts teachng how common web vulnerabilities arise from insecure application design, poor input handling, and misconfigured services,

The excersices simulate realistic attack surfaces found in web applications and require analyzing  both client-side and server-side behavior.
